广西北海严查旅游餐饮宰客乱象

北海风光

１０月１日，广西北海市旅游局联合工商、公安、质监等执法部门，对该市海鲜大排档进行抽查时发现：旅游餐饮市场存在着诸多宰客乱象。

这些乱象包括：

鱼目混珠，“凡品”变“珍品”。来自东北的游客正在北海市银滩大道上的一家大排档就餐时，恰巧遇上了前来检查的执法部门。他们点的高档海鲜，等到上桌后才发现是普通海鲜，仅一条鱼的价格就相差了３００元钱。

价格飙升几十倍。检查组在北海景区附近的渔村检查时发现，一些海鲜大排档的螺类价格，前几天还只是６－１０元／公斤，如今在大排档的价格竟然达到了惊人的近１００元／公斤，平日里价格不过６０－７０元／公斤的石斑鱼，价格也飙升到近６００元／公斤。

短斤少两情况严重。在一些海鲜大排档，执法人员用砝码对大排档“公平秤”进行检测时，发现大排档“公平秤”比砝码多出６０－７０克的情况。按照有关规定，５公斤至２０公斤的商品重量只允许存在１０克的误差。

北海市旅游局负责人表示，将对这些欺客、宰客的违法经营现象从重处罚，并严格执行“三罚退市制”等相关政策，对全市海鲜大排档进行规范管理。
